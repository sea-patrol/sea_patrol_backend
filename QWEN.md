# Sea Patrol Backend

## Обзор проекта

**Sea Patrol Backend** — это серверная часть многопользовательской морской игры, построенная на Spring Boot WebFlux. Проект предоставляет:

- **Многопользовательский режим в реальном времени** через WebSocket-соединения
- **Аутентификация пользователей** с помощью JWT-токенов
- **Управление игровыми комнатами** для многопользовательских сессий
- **Физическая симуляция** с использованием LibGDX и Box2D
- **Реактивная архитектура** на базе Spring WebFlux и Project Reactor

### Технологический стек

| Компонент | Технология |
|-----------|------------|
| **Фреймворк** | Spring Boot 3.5.6 (WebFlux) |
| **Язык** | Java 24 |
| **Сборка** | Gradle 8.14.3 (Kotlin DSL) |
| **Безопасность** | Spring Security + JWT (jjwt 0.9.1) |
| **Игровой движок** | LibGDX 1.12.1 + Box2D |
| **ИИ** | gdx-ai 1.8.2 |
| **Утилиты** | Lombok, MapStruct |

### Архитектура

```
src/main/java/ru/sea/patrol/
├── SeaPatrolApplication.java    # Точка входа приложения
├── config/                       # Конфигурация безопасности и WebSocket
├── controller/                   # REST и WebSocket контроллеры
├── dto/                          # Объекты передачи данных
│   ├── auth/                    # DTO аутентификации
│   └── websocket/               # DTO WebSocket-сообщений
├── entity/                       # JPA-сущности (UserEntity, UserRole)
├── errorhandling/                # Глобальная обработка ошибок
├── exception/                    # Пользовательские исключения
├── handler/                      # WebSocket-обработчики
├── mapper/                       # MapStruct-мапперы
├── repository/                   # Репозитории (in-memory)
├── security/                     # JWT и утилиты аутентификации
└── service/                      # Бизнес-логика
    ├── chat/                    # Сервис чата
    └── game/                    # Управление комнатами и игроками
```

## Сборка и запуск

### Требования

- **Java 24** или совместимый JDK
- **Gradle 8.14.3** (wrapper включён)

### Команды

```bash
# Сборка проекта
./gradlew build

# Запуск приложения
./gradlew bootRun

# Запуск тестов
./gradlew test

# Очистка и сборка
./gradlew clean build
```

### Конфигурация

Основная конфигурация находится в `src/main/resources/application.yaml`:

```yaml
spring:
  application:
    name: sea.patrol

jwt:
  secret: <ваш-секретный-ключ>
  expiration: 3600
  issuer: seapatrol
```

### Настройки CORS

Приложение настроено на приём соединений с:
- `http://localhost:5173` (Vite dev server)
- `http://localhost:4173` (Vite preview server)

## Соглашения разработки

### Стиль кода

- **Lombok** используется для уменьшения шаблонного кода (`@Data`, `@Builder`, `@RequiredArgsConstructor`)
- **MapStruct** для маппинга объектов (безопасность на этапе компиляции)
- Реактивное программирование с **Project Reactor** (`Mono`, `Flux`)

### Тестирование

- **JUnit 5** с Spring Boot Test
- Тесты расположены в `src/test/java/ru/sea/patrol/`
- Запуск тестов: `./gradlew test`

### Ключевые возможности

1. **Процесс аутентификации**
   - JWT-аутентификация
   - Публичные маршруты: `/api/v1/auth/signup`, `/api/v1/auth/login`
   - Все остальные маршруты требуют аутентификации

2. **WebSocket-протокол игры**
   - Обработка ввода игрока в реальном времени
   - Обновления состояния игры
   - Обмен сообщениями в чате
   - События присоединения/выхода игроков

3. **Игровая механика**
   - Корабли игроков со здоровьем, скоростью, позицией
   - Физическая симуляция через Box2D
   - Эффекты ветра на геймплей
   - Подбор комнат (matchmaking)

## Структура проекта

| Директория | Назначение |
|------------|------------|
| `src/main/java` | Исходный код приложения |
| `src/main/resources/static` | Фронтенд-ресурсы (React-приложение) |
| `src/main/resources/templates` | HTML-шаблоны |
| `src/test/java` | Юнит- и интеграционные тесты |

## Примечания

- Проект использует **in-memory репозитории** (база данных не настроена)
- Фронтенд (React) обслуживается из директории `static/`
- WebSocket-эндпоинт обрабатывает коммуникацию в реальном времени
- Box2D natives включены для десктопных платформ
